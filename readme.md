<h3>PostFormat 文章形式</h3>
<ul>
<li>当前版本: 1.0.0<br /></li>
<li>更新时间: 2012.7.7</li>
</ul>
<h3>使用方法</h3>
<ol>
<li>解压文件, 将插件复制到<code>build/usr/plugins/</code>文件夹下, 然后激活;</li>
<li>本插件提供了6种文章形式:<code>标准</code>, <code>短语</code>, <code>视频</code>, <code>音频</code>, <code>图像</code>, <code>链接</code>;</li>
</ol>
